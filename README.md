# March Madness Predictive Model

## Overview

This project is an NCAA March Madness prediction model built in Microsoft Excel.

The model predicts tournament outcomes using historical and performance-based team metrics, then outputs winners in chronological tournament order.

---

## Key Features

* Predicts winners for all 67 NCAA tournament games
* Uses structured tournament logic with chronological matchup flow
* Calculates team strength using multiple performance metrics
* Compares predicted results vs actual tournament outcomes
* Identifies key metrics associated with team success

---

## Model Structure

The model includes:

* **Team Database** — Contains all tournament teams
* **Performance Metrics** — Offensive, defensive, and ranking data
* **Matchup Engine** — Calculates predicted winners
* **Model Results** — Displays predicted outcomes chronologically
* **Actual Results** — Real tournament results (as of March 23, 2026)
* **Model Analysis** — Identifies which metrics drive winning

---

## Key Metrics Used

* Offensive Rating
* Defensive Rating
* Strength of Schedule
* Turnover Rate
* Rebounding Margin
* Seed Ranking

---

## Example Output

The model produces:

* Full predicted tournament bracket
* Chronological game results
* Prediction accuracy tracking
* Statistical analysis of winning team traits
